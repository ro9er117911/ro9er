# ro9er

React + Vite personal portfolio for Roger.

Target deployment:

- GitHub Pages project site: `https://ro9er117911.github.io/ro9er/`
- Repository: `https://github.com/ro9er117911/ro9er`

Local development:

```bash
npm install
npm run dev
```

Production build / preview:

```bash
npm run typecheck
npm run build
npm run preview
```

Open the local URL printed by Vite. The production build writes GitHub Pages-ready files to the repo root.
